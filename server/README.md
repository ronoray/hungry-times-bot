# Hungry Times Sales Bot

Sales and marketing co-pilot bot for Hungry Times restaurant.

## Features
- AI-powered marketing advice
- Sales analytics via Telegram
- Campaign creation assistance
- Website analysis

## Setup

1. Copy `.env.example` to `.env`
2. Fill in your API keys
3. Deploy via ops panel webhook

## Commands

- `/test` - System health check
- `/analytics` - Last 7 days sales data
- `/reset` - Clear conversation

## Deployment

Deploys automatically via webhook when pushing to main branch.
